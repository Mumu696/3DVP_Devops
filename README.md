# 3DVP_Devops

## API Documentation

### GET /api/services
Returns the list of available services.

### GET /api/status
Returns the application status.

## Installation
1. Clone the repository
2. Run `npm install`
3. Start the server with `npm start`

## Development
Use `npm run dev` for development with hot-reload.
